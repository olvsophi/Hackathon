# Secure Search

O **Secure Search** é uma plataforma colaborativa desenvolvida para ajudar usuários a identificar sites suspeitos, golpes, fake news e possíveis ameaças na internet. Além de consultar denúncias, qualquer usuário pode contribuir com novos relatos, fortalecendo a conscientização sobre segurança digital.

## Funcionalidades

- Pesquisa de denúncias sobre sites e links.
- Cadastro de novas denúncias.
- Sistema de login e cadastro com Firebase Authentication.
- Listagem de denúncias com paginação.
- Conteúdo educativo sobre cidadania online e segurança digital.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Firebase (Authentication e Firestore)

## Design

A interface foi projetada no **Figma**, utilizando tons de azul para transmitir segurança e elementos no estilo **Liquid Glass**, proporcionando uma experiência moderna e intuitiva.
